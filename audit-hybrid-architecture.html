📋 Compliance Rules

Supabase-First Hybrid Architecture Requirements:

  1. ✅ Read Pattern: Try Supabase first, fallback to localStorage only if Supabase fails
  2. ✅ Write Pattern: Write to Supabase first, then cache in localStorage
  3. ✅ Cache Pattern: Replace localStorage with Supabase data (not merge) after successful Supabase read
  4. ✅ Error Handling: Graceful fallback to localStorage if Supabase unavailable
  5. ❌ Anti-Pattern: Only using localStorage without checking Supabase
  6. ❌ Anti-Pattern: Writing only to localStorage without writing to Supabase
