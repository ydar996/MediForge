Bulk Import Patients

Upload a CSV or Excel file to register many patients at once. Only first name or last name is required per row — leave other fields empty and complete them later in the patient chart.

Manage Patients Download Excel/CSV template
Preferred template: Use the downloadable CSV (open in Excel, fill rows, save as CSV or upload .xlsx).
Patient IDs: If your clinic already uses file numbers, put them in the Patient ID column and enable Keep existing Patient IDs below. Leave blank to use your organization’s auto-numbering (e.g. MFA-SC0001).
Semi-structured files: Column headers are matched flexibly (e.g. “Surname”, “PHN”, “DOB”). Medications and allergies can be semicolon-separated in one cell.
