??? Clear localStorage

This will clear all patient data from localStorage, forcing MediForge to load fresh data from Supabase.

?? Warning: This will clear LOCAL data only. Supabase data will remain intact.

? Success!

localStorage has been cleared.

Now go to MediForge and refresh the page.

The app will load fresh data from Supabase with the correct DOB values.

function clearLocalStorage() { // Get user data first const user = JSON.parse(localStorage.getItem("user") || "{}"); const org = user.org || 'mecure'; // Clear prefixed patient data localStorage.removeItem(`${org}_patients`); localStorage.removeItem('patients'); // Also clear mecure_patients specifically localStorage.removeItem('mecure_patients'); console.log('Cleared localStorage for:', org); // Show success message document.getElementById('success').style.display = 'block'; }
