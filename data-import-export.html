Data Import & Export

Transfer data between systems, bulk import patients, and ensure data portability

ℹ Important Information:
  • Always create a backup before importing data
  • CSV imports will add to existing data, not replace it
  • Duplicate patient IDs will be automatically skipped
  • You can undo recent imports by restoring a backup

📥 Import Patients from CSV

Upload a CSV file to bulk import patients. The CSV should include columns for: First Name, Last Name, DOB, Gender, and optionally other demographic fields.

Supported formats: .csv, .txt
Required columns: First Name, Last Name, DOB
Optional columns: Gender, Phone, Email, Address, City, State, Country, etc.

📥 Export Patients to CSV

Download all patients as a comprehensive CSV file. This includes all demographic data, medical history counts, and contact information.

Includes: Demographics, contact info, insurance, medical record counts
Use for: Reporting, analysis, migration to other systems

💾 Complete System Backup

Create a complete backup of ALL data including patients, appointments, prescriptions, documents, and user settings. This is the most comprehensive backup option.

Includes: Everything - patients, appointments, users, documents, settings
Use for: Disaster recovery, system migration, archival

📁 Restore from Backup

Restore a previous backup file. This will REPLACE all current data with the data from the backup file.

⚠ Warning: This will overwrite all current data!
Recommendation: Create a backup first before restoring

📄 HL7 Format Export

Export patient data in HL7 format for interoperability with other healthcare systems. Available for individual patients on their details page.

Format: HL7 v2.5 ADT messages
Use for: Hospital integration, health information exchange

🗃 Storage Information

Check how much browser storage you're using and identify large data items.

Limit: ~5-10MB for localStorage
Tip: Create regular backups and archive old patient data

📋 CSV Import Template (Column Headers): 
First Name,Last Name,DOB,Gender,Phone,Email,Address Line 1,City,State,Country

Copy this header row into a spreadsheet, add your patient data below it, and save as CSV.
