Migration Overview

This tool will migrate patient demographic data from localStorage to Supabase.

What it does:

  • Reads all patients from localStorage
  • Updates their Supabase records with missing demographic fields
  • Handles data conflicts gracefully
  • Provides detailed logging and progress tracking
