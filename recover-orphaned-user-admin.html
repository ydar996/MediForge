🔧 Recover Orphaned User

This tool helps recover users who have Auth accounts but no profiles in the users table.

You can find this in Supabase Dashboard → Organizations table

Option 1: Recover with Password

Use this if the user remembers their password.

Option 2: Reset (Forgotten Password)

Use this if the user forgot their password. This will delete the orphaned Auth user so they can register fresh.

Enter just the username - the email will be constructed automatically
Leave empty if you only have username
