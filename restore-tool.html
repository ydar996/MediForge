MediForge Restore Tool

Restore from backup if needed

WARNING - USE WITH CAUTION!

This will REPLACE all current data with backup data!

Only use this if:

  • Something went wrong during migration
  • Data was corrupted or lost
  • You need to roll back to a previous state

This cannot be undone! Make sure you have a current backup first!

How This Works:

  1. Upload your backup JSON file (from Downloads or Google Drive)
  2. Preview the backup to verify it's correct
  3. Click "RESTORE ALL DATA" button
  4. All data will be restored to localStorage
  5. Refresh the app to see restored data
